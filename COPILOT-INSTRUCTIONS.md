# FerroCheck - Instrucciones del Proyecto

## Objetivo

FerroCheck es un sistema para automatizar el proceso de verificación de inventario ferroviario de VASCOR.

## Arquitectura

- MVC
- Service Layer
- Repository Pattern

## Tecnologías

- PHP 8.3
- HTML5
- CSS3
- JavaScript ES6
- MariaDB
- Composer
- Git
- Docker

## Convenciones

- Variables en español.
- Clases en PascalCase.
- Métodos en camelCase.
- CSS y JS en kebab-case.
- Código limpio.
- SOLID cuando aplique.
- No duplicar código.

## Regla más importante

Antes de escribir código, analiza la arquitectura existente.

Nunca modifiques archivos que no fueron solicitados.

Si detectas una mejor solución, explícala antes de implementarla.

## Visión del proyecto

En futuras versiones FerroCheck tendrá:

- OCR
- Inteligencia Artificial
- API REST
- PWA
- Integración con Immich
- Captura de fotografías
- Evidencias automáticas
- Reportes PDF
- Dashboard
